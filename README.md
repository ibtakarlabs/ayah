# AMK Artistry - Professional Makeup Artist Website

A beautiful, modern website for AMK Artistry featuring professional makeup services by Ayah Kawsara.

## Features

- Responsive design optimized for all devices
- Smooth animations with Framer Motion
- Optimized AVIF images for fast loading
- Clean, girly and glam aesthetic
- Professional portfolio gallery
- Contact form for bookings
- Social media integration

## Tech Stack

- React 18
- Vite
- Framer Motion
- Custom CSS with modern design

## Getting Started

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

The site will be available at `http://localhost:3000`

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Design Features

- **Color Palette**: Rose gold, soft pink, cream, and deep plum
- **Typography**: Cormorant Garamond (headings) & Montserrat (body)
- **Animations**: Smooth fade-ins, parallax effects, and hover animations
- **Layout**: Modern grid system with responsive breakpoints

## Sections

1. **Hero** - Eye-catching landing section with call-to-action
2. **About** - Introduction to Ayah with experience badge
3. **Services** - Four service cards (Bridal, Glam, Editorial, Events)
4. **Gallery** - Filterable portfolio with Instagram integration
5. **Contact** - Contact form with social links
6. **Footer** - Site navigation and social media links
